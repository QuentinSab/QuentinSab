## Projects

### ETL Pipeline – Books to Scrape

My first Python project.
The objective of this project was to build an automated ETL (Extract, Transform, Load) pipeline capable of collecting data from an online bookstore. The application extracts product information from web pages, transforms the data into a structured format, and exports the results to CSV files for further analysis.

The project evolved from scraping a single product page to processing entire categories and eventually the complete website. It also includes the automated download and storage of product images.

[OC-P2 Repository](https://github.com/QuentinSab/OC-P2)
